# Recipes App project

This project is a conversation application. It is possible to create new conversations with any registered user.

## Technologies used
  ### Back-end
    - Node.js
    - Express
    - TypeScript
    - Web Socket
    - MySQL
    - Sequelize
    - Cors
    
  ### Front-end
    - React.js
    - CSS3
    - Axios
    - ContextAPI
 
  ### Tools
    - Docker
    - Git
